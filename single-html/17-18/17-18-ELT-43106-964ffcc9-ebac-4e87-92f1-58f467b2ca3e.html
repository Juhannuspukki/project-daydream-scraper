Miten kehittäisit opintojaksoa?
Opettajan vastaus :
It is true that the lecture notes need to be improved and formatted in unified style. The current trend is to have one-period courses, but fewer of those at the same time. If this is not done systematically, the students' load becomes high in the 4th period, since the projects of the longer courses usually need to be done during period 4.
