Nykyinen vastaus
Thank you for choosing this course and for the valuable feedback. As mentioned course is demanding but also rewarding if you can invest some time into it. This requires good planning and organization within the project group. Some succeeded better in this but in general all the results are pretty good. Time you spend on the project work can be unlimited if you like. But to help students to prioritize we have given the grading basis so you can decide on what you should spend time and what thing can be left to smaller attention. Exam is demanding and has been used as means to verify each students learns the key topics. For next implementation we'll check if it could be made more focused. Of course the garding basis need to be changed too.
