Nykyinen vastaus
This year's implementation of the course had much more programming exercises than the previous course implementations. Still, based on the student feedback, even more programming and FPGA related exercises would be preferred. The autumn 2019 implementation of the course will introduce new "optimization of neural networks" related content, and we'll ensure that both the new and existing content is supported by a higher count of programming and FPGA exercises.

Miten kehittäisit opintojaksoa?
Opettajan vastaus :
Thanks for the constructive comments! - Regarding the student's wish for a comprehensive application use case that would need all the taught methods -- it would be really hard to find such a use case. The taught methods and techniques should be regarded as a toolbox of approaches for tackling various design and implementation related challenges. Nevertheless, we'll try to come up with exercises that require simultaneous application of several techniques. - We'll see that the lectured contents come closer to practical applications and to the pen&paper exercises - The time frame for handing back exercises really varied quite a lot this year, and the issue will be addressed next year - Also, we'll pay attention to the clarity of FPGA exercises. Unfortunately, FPGA design and synthesis requires a lot of parameters that to a great extent go beyond the scope of this course. In this sense students just need to take some FPGA parameters as given, and hopefully take an FPGA-centered course where the FPGA related content is then explained in detail.
